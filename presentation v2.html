Why AI is Essential for This Solution

This problem is fundamentally impossible to solve without AI. Every core feature requires intelligent visual understanding that only AI can provide.

Virtual Try-On Requires Visual Intelligence

Placing a garment on a person's body while preserving their exact face, skin tone, body shape, and pose requires deep understanding of human anatomy, fabric physics, and lighting. No rule-based system or simple image overlay can achieve this — only generative AI models like Gemini can produce photorealistic results that look natural.

Outfit Conflict Resolution Needs Reasoning

When a user wearing a dress tries on pants, AI must reason: "remove the dress, add the pants, and generate a matching top." This requires understanding of clothing categories, fashion coordination, and compositional reasoning that only AI can perform.

Product Classification at Scale

Online retailers have millions of products. AI classifies each into garment categories (upper body, full body, footwear, cosmetics) in real-time, determining the correct try-on strategy automatically. Manual categorization is impossible at this scale.

AI Agent for Natural Language Shopping

Playwright-based smart search doesn't just search keywords — it browses retailers like a human, navigating pages, applying filters, scrolling results, and extracting product data. This agentic behavior requires AI that understands web interfaces and user intent.

Identity-Preserving Profile Generation

Generating 3 professional model poses from 5 casual user photos — while preserving exact facial features, body proportions, and skin tone — requires multi-reference AI that learns a person's identity from limited examples.

Video Animation from a Single Image

Transforming a static try-on photo into a 6-second video with natural movement (weight shifts, fabric flow, subtle poses) requires AI that understands physics, human motion, and fashion presentation. No traditional animation tool can do this.

Why a Chrome Extension?

We can't modify retailer websites directly — a Chrome extension is the only way to enhance the shopping experience without platform access.

The Technical Reality

Online retailers are closed platforms. We cannot inject features into their product pages from a standalone website. A Chrome extension uses content scripts to seamlessly overlay our AI-powered UI directly onto existing product pages — the "Try It On" button appears right where the user is already shopping.

Zero Friction Adoption

Users don't leave the retailer site. They don't open a second app or website. The try-on experience is embedded into their existing shopping workflow. Install once, and every product page becomes a virtual fitting room automatically.

Multi-Retailer Universal Platform

This extension works across Amazon, SHEIN, Temu, and Google Shopping — providing a universal try-on experience regardless of which retailer the user prefers. One extension, all stores.

Smart Try-On Pipeline

5-step AI orchestration with intelligent outfit conflict resolution

Pipeline Steps

1. Product Analysis Gemini 2.5 Flash
Classifies garment type, color, category. Detects if image has a model.
2. Person Detection & Extraction Gemini 2.5 Flash Image
If product image has a person, extracts garment. Gemini 2.5 Flash Image for background removal.
3. Outfit Classification Gemini 2.5 Flash
Analyzes what user is wearing to detect outfit conflicts.
4. Smart Prompt Generation
Context-aware instructions for garment + outfit conflict resolution.
5. Virtual Try-On Gemini 3.1 Flash Image
Photorealistic result preserving face, body, skin tone. Chosen for accuracy and speed.

Conflict Resolution Examples

Voice Agent Architecture

Vertex AI backend proxy with WebSocket — server-side authentication for Gemini Live API

WebSocket Proxy Architecture

Browser connects to Cloud Run backend via WebSocket. Backend proxies to Gemini Live API via Vertex AI using @google/genai SDK with server-side authentication.

Flow: Browser → WebSocket → Cloud Run → Vertex AI → Gemini Live API
Auth: Service account with roles/aiplatform.user IAM role

Giselle's Persona & Voice

Distinct warm persona with Aoede voice. Native audio generation (not text-to-speech).

12 Tools: smart_search, try_on, build_outfit, recommend_items, save_favorite, animate_tryon, save_video, download, send_tryon, select_search_item, suggest_item, select_outfit_items
Features: Input/output audio transcription, barge-in support, echo cancellation

How It Works

Client-Side Tool Execution

When Giselle calls a tool, the client (popup.js) executes it locally — search, try-on, outfit build, etc. Results flow back to Giselle via tool responses.

Echo Cancellation & Barge-In

Energy-threshold gating prevents echo-triggered self-interruption while preserving natural barge-in. Grace period protects greeting from being cut off.

Vision-Based Recommendations

Captures the user's screen and photo, sends to Gemini 3 Flash Preview with thinking mode for personalized styling analysis.

Completion Notifications

Content scripts notify Giselle when try-on, video generation, or outfit building completes, so she can report results back to the user naturally.

Native Audio Generation

Gemini Live 2.5 Flash Native Audio generates speech directly (not text-to-speech), producing natural, expressive voice output with the Aoede voice.

Get started in under 5 minutes

Installation Steps

1. Download Extension
Clone the repo or download the extension folder from GitHub.
2. Load in Chrome
Go to chrome://extensions, enable Developer Mode, click "Load unpacked", select the extension/ folder.

Click the extension icon, sign up with email. Firebase Auth handles verification.

Upload 5 photos (3 full-body + 2 face). AI generates 3 poses. Browse any retailer and try on!
